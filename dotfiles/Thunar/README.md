# Thunar

## Описание

Конфигурация для Thunar — файлового менеджера для Xfce. Содержит настройки:

- Пользовательские действия (uca.xml) — контекстное меню с дополнительными командами
- Горячие клавиши (accels.scm) — пользовательские комбинации клавиш

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/Thunar/uca.xml` — пользовательские действия (контекстное меню)
- `~/.config/Thunar/accels.scm` — горячие клавиши

## Структура пакета

```
dotfiles/Thunar/
└── .config/
    └── Thunar/
        ├── uca.xml
        └── accels.scm
```

## Применение

```bash
./stow-init.sh --apply Thunar
```

## Особенности

- **Пользовательские действия**: дополнительные команды в контекстном меню
- **Горячие клавиши**: настройка комбинаций клавиш для быстрого доступа к функциям
- Конфигурация применяется автоматически после перезапуска Thunar
