# Nitrogen

## Описание

Конфигурация для Nitrogen — утилиты для установки обоев рабочего стола в X11. Содержит настройки:

- Геометрия окна приложения
- Настройки просмотра обоев (список, рекурсивный поиск, сортировка)
- Директории с обоями

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/nitrogen/nitrogen.cfg` — основной файл конфигурации
- `~/.config/nitrogen/bg-saved.cfg` — сохраненные обои и их позиции

## Структура пакета

```
dotfiles/nitrogen/
└── .config/
    └── nitrogen/
        ├── nitrogen.cfg
        └── bg-saved.cfg
```

## Применение

```bash
./stow-init.sh --apply nitrogen
```

## Особенности

- **Режим просмотра**: список (list)
- **Рекурсивный поиск**: включен
- **Сортировка**: по алфавиту (alpha)
- **Директория обоев**: `/usr/share/wallpapers/`
- **Геометрия окна**: настроена под конкретное разрешение экрана

## Использование

После применения конфигурации можно запустить Nitrogen:

```bash
nitrogen
```

Для автоматической установки обоев при входе в систему можно добавить в автозапуск (например, в i3):

```i3
exec --no-startup-id nitrogen --restore
```
