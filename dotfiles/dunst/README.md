# Dunst

## Описание

Конфигурация для Dunst — легковесного демона уведомлений для Linux. Содержит настройки:

- Внешний вид уведомлений (размеры, позиция, прозрачность)
- Цветовая схема и шрифты
- Правила для различных типов уведомлений
- Поведение и таймауты

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/dunst/dunstrc` — основной файл конфигурации

## Структура пакета

```
dotfiles/dunst/
└── .config/
    └── dunst/
        └── dunstrc
```

## Применение

```bash
./stow-init.sh --apply dunst
```

## Особенности

- **Шрифт**: FiraCodeNerdFont-Regular 16
- **Цвет рамки**: #4287f5 (синий)
- **Размеры**: ширина до 400px, высота до 200px
- **Позиция**: следует за курсором мыши
- **Markup**: поддержка полной разметки (full)
- **Сортировка**: по приоритету (urgency)

## Перезапуск

После применения конфигурации необходимо перезапустить dunst:

```bash
pkill dunst && dunst &
```

Или перезагрузить конфигурацию:

```bash
killall -SIGUSR1 dunst
```
