# i3 Window Manager

## Описание

Конфигурация окнного менеджера i3 (i3wm). Содержит настройки:

- Горячие клавиши для управления окнами и рабочими пространствами
- Автозапуск приложений (copyq, dunst, nm-applet, power-manager и др.)
- Настройки внешнего вида (цвета, границы, gaps)
- Правила размещения окон (floating mode для определенных приложений)
- Интеграция с i3bar для отображения статуса

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/i3/config` — основной файл конфигурации i3

## Структура пакета

```
dotfiles/i3/
└── .config/
    └── i3/
        └── config
```

## Применение

```bash
./stow-init.sh --apply i3
```

## Особенности

- Модификатор: `Super` (Mod4)
- Терминал по умолчанию: `kitty`
- Браузер: `firedragon`
- Файловый менеджер: `thunar`
- Редактор: `geany`
- Скриншоты: `flameshot`
