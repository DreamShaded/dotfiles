# my-snippets

Единый пакет пользовательских сниппетов (алиасы/функции/ENV), который должен работать и в bash, и в fish.

## Структура

- **bash**: `~/.r.snippets/*.sh` (подхватывается из `my-settings/shell/.bashrc`)
- **fish**: `~/.r.snippets/fish/*.fish` (подхватывается из `my-settings/fish/.config/fish/conf.d/r-snippets.fish`)

## Как добавлять новые сниппеты

- **bash-специфично**: добавляй новый файл `*.sh` в `.r.snippets/`
- **fish-специфично**: добавляй новый файл `*.fish` в `.r.snippets/fish/`
- **хочется “одинаково везде”**: почти всегда нужны 2 файла (fish и bash разные по синтаксису)
  - Практика, чтобы всё равно был “один источник истины”: выноси общую логику в `~/.local/bin/...` (скрипты),
    а в shell оставляй тонкие обёртки.
