# Stalonetray

## Описание

Конфигурация для Stalonetray — легковесного системного трея (system tray) для оконных менеджеров без встроенного трея. Содержит настройки:

- Позиция и размер трея
- Внешний вид (цвет фона, размер иконок)
- Поведение (sticky, dock mode)

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/stalonetray/stalonetrayrc` — основной файл конфигурации

## Структура пакета

```
dotfiles/stalonetray/
└── .config/
    └── stalonetray/
        └── stalonetrayrc
```

## Применение

```bash
./stow-init.sh --apply stalonetray
```

## Особенности

- **Геометрия**: 5x1 иконок, позиция 1820-0 (правый верхний угол)
- **Фон**: #0D0D16 (темный)
- **Размер иконок**: 20px
- **Режим**: dock (работает как dock-окно)
- **Sticky**: включен (прилипает ко всем рабочим пространствам)
- **Прозрачность**: отключена

## Использование

После применения конфигурации можно запустить stalonetray:

```bash
stalonetray
```

Для автозапуска добавьте в конфигурацию window manager (например, i3):

```i3
exec --no-startup-id stalonetray
```

## Примечания

- Stalonetray полезен, когда window manager (например, i3) не имеет встроенного трея
- Геометрия настроена для конкретного разрешения экрана, может потребоваться корректировка под ваше разрешение
