# Redshift

## Описание

Конфигурация для Redshift — утилиты для регулировки цветовой температуры экрана в зависимости от времени суток. Снижает нагрузку на глаза в вечернее время, делая экран "теплее" (более красным).

Содержит настройки:
- Цветовая температура для дня и ночи
- Географическое местоположение (широта/долгота) для определения времени заката/рассвета
- Режим работы (gamma, brightness)

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/redshift/redshift.conf` — основной файл конфигурации

## Структура пакета

```
dotfiles/redshift/
└── .config/
    └── redshift/
        └── redshift.conf
```

## Применение

```bash
./stow-init.sh --apply redshift
```

## Особенности

- Автоматическое определение местоположения или ручная настройка координат
- Настройка температуры днем (обычно 5500K - дневной свет) и ночью (обычно 3700K - теплый свет)
- Регулировка гаммы и яркости
- Автоматический переход между режимами в зависимости от времени суток

## Использование

После применения конфигурации можно запустить redshift:

```bash
redshift
```

Для автозапуска можно использовать systemd user service:

```bash
systemctl --user enable redshift.service
systemctl --user start redshift.service
```

Или добавить в автозапуск window manager (например, в i3):

```i3
exec --no-startup-id redshift
```

## Примечания

- Redshift требует либо настроенные координаты в конфигурации, либо доступ к геолокации
- На некоторых системах может потребоваться установка пакета `redshift` или `redshift-gtk` (GUI версия)
