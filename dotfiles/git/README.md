# Git

## Описание

Конфигурация для Git — системы контроля версий. Содержит настройки:

- Информация о пользователе (имя, email)
- Полезные алиасы для команд Git
- Настройки поведения (автоматический rebase при pull, default branch)
- Настройки безопасности и автозамены

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.gitconfig` — основной файл конфигурации Git
- `~/.gitignore_global` — глобальный файл с игнорируемыми паттернами

## Структура пакета

```
dotfiles/git/
├── .gitconfig
└── .gitignore_global
```

## Применение

```bash
./stow-init.sh --apply git
```

## Особенности

- **Алиасы**: `st` (status), `co` (checkout), `cm` (commit -m), `rbi` (rebase -i) и др.
- **Pull**: автоматический rebase при pull (можно переопределить)
- **Default branch**: `master`
- **Autocrlf**: `input` (для Linux)

## Настройка

⚠️ **Важно**: После применения пакета необходимо отредактировать `~/.gitconfig` и указать свои данные:
- `user.email`
- `user.name`
