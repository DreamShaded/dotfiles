# i3status

## Описание

Конфигурация для i3status — утилиты, которая показывает статус системы в строке состояния i3bar. Содержит настройки для отображения:

- Свободное место на диске (`/`)
- Состояние Wi-Fi соединения
- Заряд батареи и время работы
- Температура процессора
- Уровень громкости
- Дата и время (локальное время)

Использует формат вывода i3bar с поддержкой Pango markup для форматирования текста и цветов.

## Расположение файлов

После применения через `stow-init.sh`, конфигурация будет установлена в:

- `~/.config/i3status/config` — основной файл конфигурации i3status

## Структура пакета

```
dotfiles/i3status/
└── .config/
    └── i3status/
        └── config
```

## Применение

```bash
./stow-init.sh --apply i3status
```

## Особенности

- **Интервал обновления**: 1 секунда
- **Формат вывода**: i3bar с Pango markup
- **Порядок модулей**: диск, Wi-Fi, батарея, температура CPU, громкость, время
- **Локализация**: немецкая (de_DE.UTF-8) для даты и времени
- **Цвета**: белый текст (#FFFFFF) для всех модулей

## Примечания

- Путь к датчику температуры CPU может потребовать настройки под вашу систему (`/sys/class/hwmon/hwmon3/temp1_input`)
- Путь к батарее настроен на `/sys/class/power_supply/BAT0/uevent`
- Модуль громкости использует ALSA Master mixer
